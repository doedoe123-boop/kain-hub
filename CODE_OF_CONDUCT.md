# Code of Conduct

We value a welcoming, inclusive, and respectful community. By participating in **NegosyoHub**, you agree to follow this Code of Conduct.

## Our Standards

- Be **respectful** and considerate in all interactions.
- Use **inclusive language**; avoid discriminatory, offensive, or harmful comments.
- Be **constructive** when giving feedback.
- Respect the **privacy** of others.
- Report unacceptable behavior to the maintainers.

## Enforcement

Project maintainers may remove or block participants who violate this Code of Conduct. Serious issues may be reported to GitHub support.

## Contact

If you experience or witness misconduct, please contact the maintainers via asum.nelson@gmail.com or open a confidential issue labeled `security`.

---

By participating, you agree to uphold these standards and help keep NegosyoHub a safe and welcoming community.
